# 10. Esame Finale di Minikube

Complimenti! Sei arrivato alla fine del corso. In questo esame verificheremo che tu sappia gestire un cluster Minikube completo.

## La Sfida di Oggi
Dovrai simulare l'avvio di un ambiente di sviluppo complesso:
1. Crea un cluster con un profilo specifico.
2. Imposta risorse adeguate (CPU/RAM).
3. Abilita gli strumenti di gestione (Ingress, Dashboard).
4. Verifica che tutto sia pronto per ospitare le tue applicazioni.

## Obiettivi dell'Esame
- Saper configurare Minikube per diversi scenari.
- Conoscere i meccanismi di esposizione dei servizi.
- Saper gestire cluster multi-nodo.
- Risolvere i problemi di base usando i comandi di status e log.
