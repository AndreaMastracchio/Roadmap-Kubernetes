# Corso Minikube: Kubernetes in Locale

Benvenuto nel corso dedicato a **Minikube**, lo strumento standard per eseguire un cluster Kubernetes a nodo singolo sul tuo computer locale.

## Cosa imparerai
- Come installare e configurare Minikube.
- Gestione dei profili e dei driver (Docker, VirtualBox, SSH).
- Utilizzo degli Add-ons (Ingress, Dashboard, Metrics Server).
- Accesso ai servizi locali e debugging.

## Prerequisiti
- Conoscenza base dei container (Docker).
- Terminale/Shell installata.
