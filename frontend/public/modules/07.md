# Modulo 7: ☸️ Helm (Il Gestore dei Pacchetti)

Helm è per Kubernetes quello che `apt` è per Ubuntu o `npm` per Node.js.

## 1. Chart
Un pacchetto Helm si chiama **Chart**. È una cartella contenente file YAML con dei template (solitamente usando il linguaggio di templating di Go).

## 2. Perché Helm?
Invece di avere centinaia di file YAML duplicati per ogni ambiente (dev, staging, prod), Helm permette di avere un unico set di template e di passare i valori diversi in un file `values.yaml`.

## Esempio struttura Chart
```text
mychart/
  Chart.yaml          # Metadati della chart
  values.yaml         # Valori di default
  templates/          # I file YAML con i placeholder
    deployment.yaml
    service.yaml
```

## Comandi principali
- `helm install <nome-release> <nome-chart>`: Installa un'applicazione.
- `helm upgrade <nome-release> <nome-chart>`: Aggiorna un'applicazione esistente.
- `helm repo add <nome> <url>`: Aggiunge un repository di chart (es. Bitnami).
